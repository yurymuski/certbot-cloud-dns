# certbot-cloud-dns

Bash scripts for issuing Lets encrypt wildcard certs based on DNS challenges

Requirements: docker

Supported cloud DNS:
 - [Digital Ocean](digitalocean/README.md)
 - [Azure](azure/README.md)
 - [Cloudflare](cloudflare/README.md)
 - [AWS](aws/README.md)
 - [Yandex Cloud](yandex/README.md)
